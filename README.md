# Gestión de turnos médicos  

Gestión de turnos médicos es un CRUD que realicé con la tecnología de **Spring Boot** con el objetivo de practicar mis skills y mostrar mis conocimientos.  
A futuro pretendo agregar un frontend con React.  

## Features  

1. Clase **Usuario**  
2. Implementación de un **token JWT**  
3. Documentación de **Usuario** y testing  
4. Documentación de la implementación del **JWT** y testing del mismo.  
   Algunos fixes como que `usuarioRepository` monopolice todas las operaciones y arreglo de los tests unitarios que dejaron de funcionar después de semejante feature.  
5. Definición de **roles**:  
   - Usuario gestiona sus turnos con diferentes especialistas  
   - Médico gestiona sus turnos  
   - Administrador gestiona todos los turnos  
6. **Frontend** y diferentes vistas por rol  
7. Documentación final y testing de todo lo realizado
